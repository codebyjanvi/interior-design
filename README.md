# Acharya Woodcrafts Metal Arts Website

A modern and aesthetic business website for **Acharya Woodcrafts Metal Arts**, showcasing handcrafted wood carving, interior design services, and custom furniture products.

## Overview

This website is designed to present the company's craftsmanship and services online. It includes project galleries, product displays, and contact options so customers can easily explore the work and get in touch.

## Features

* Elegant hero section with image slider
* About section describing the company
* Services section for interior design and woodcraft work
* Project gallery showcasing completed works
* Product showcase with order option
* Instagram work preview
* Contact form with WhatsApp integration
* Google Maps direction link
* Fully responsive design
* Smooth animations and hover effects

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Google Fonts
* AOS (Animate On Scroll) animation library

## Project Structure

index.html – Main website page
style.css – Styling and layout
script.js – JavaScript for animations and interactions
images – Website images and gallery files
